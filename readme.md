# Implements PSI using FHE

## Installation
```
pip install -r requirements.txt
```

## Required dependency
1. numpy
2. Flask ( used for the demonstration )


## How to Run
1. test PSI
```
python main.py
```

2. test homomorphic operation 
```
python operate_test.py
```

3. run demonstration
```
flask run --debug
```
